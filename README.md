# homwork-react

멋쟁이 사자 리액트 과제 저장소

## 목표 : ott 사이트 메인페이지 리액트로 마크업 해보기

## 내용

1.  구조 설계 : 헤더, 상단배너, 썸네일 리스트, 푸터 총 4가지 영역 마크업

```
- <header>
- <main>
      - <section> 상단배너
      - <section> 썸네일 리스트
      - 이슈 : js에서 main 생성 후 section을 main 안에 넣기 실패
- <footer>
```

2.  CSS 연습 : 각 요소의 크기 및 색상, 레이아웃 등을 리액트에서 변경해본다.
